## Setup

1. Clone this repository
2. Add your profile photo to `assets/images/profile.jpg`
3. Add creative work images to `assets/images/`
4. Update project details in `projects.html`
5. Update coursework in `coursework.html`
6. Update creative work in `creative.html`

## Deploying to GitHub Pages

1. Go to repository **Settings**
2. Navigate to **Pages** in the sidebar
3. Under "Source", select **main** branch
4. Click **Save**
5. Your site will be live at `https://[username].github.io/keynesle/`
